# Rifa da Mia ❤️ 🐱

Este projeto foi criado com muito carinho para arrecadar fundos para a endoscopia da **Mia**, nossa gatinha que precisa de cuidados médicos.

## 🚀 Sobre o Projeto
Uma aplicação web moderna, rápida e segura para gerenciar a venda de números da rifa.

- **Frontend:** React + Vite + Tailwind CSS
- **Animações:** Framer Motion
- **Banco de Dados:** Integrado com Google Sheets (via Google Apps Script)
- **Hospedagem sugerida:** Vercel

## 🐈 Como ajudar?
Se você chegou aqui pelo código, saiba que o mais importante é a saúde da Mia. Você pode participar da rifa escolhendo seus números diretamente no site publicado!

## ⚙️ Configuração (Para Desenvolvedores)
1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `npm run dev`
3. Para publicar: Basta conectar este repositório ao **Vercel**.
